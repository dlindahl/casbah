# Casbah

A CAS server Rails Engine.

## Requirements

* Ruby on Rails 3.2 or greater
* Ruby 1.9.3+

## Installation

Add this line to your application's Gemfile:

    gem 'casbah'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install casbah
